Download this project as a .zip file

Welcome to CucumberPeople Project

CucumberPeople is an eclipse plugin based on RubyMine, Not only includes all the Cucumber Features as RubyMine published already, but also provides many enhanced project features, and It's always OPEN AND FREE

Why You Should Consider CucumberPeople ?
Take a glance some features listed as below :

  • Support cucumber language Java and Ruby
  • A powerful eclipse plugin with smart coding assistance and advanced testing and debugging features.
  • Syntax highlighting for .feature files
  • Cucumber keywords completion
  • Scenario steps completion from step definitions
  • Checking scenario steps against step definitions
  • Quick 'Go to step definition' navigation (F3 or CTRL+click)
  • Create or append step definition from usage via quick-fix
  • Console Test Runner for Cucumber scenarios
  • Integrate many Test Case Management tools with Cucumber:TestRail,TestLink,ALM(Currently It can only support TestRail )
  • Provide many Cucumber Automation frameworks templates via eclipse wizard, currently can support Java and Ruby well.
  • Using a Jenkins plugin to integrate cucumber framework with Jenkins as well.

CucumberPeople Plugin Installation Wizard

Feature - Create Customized Cucumber Project Template

After Opening your eclipse ,click the "New..." button to create a new cucumber project, which is located in the "Cucumber" category. By now I just added the Java and Ruby project template ,But We will take to support more templates with more languages...

Feature - Syntax highlighting

Keywords are highlighted so that it will be much easier to follow through your files and explain them to your business partners. Please note that the syntax highlighting is completely customizable through Window > Preferences > Cucumber > Syntax Coloring

Feature - Cucumber Structure Outline

Feature - Cucumber Formatting

just hit CTRL + SHIFT + F to have your source code pretty formatted with indentation.

Feature - Cucumber Validation

Your grammar is being validated while you type, which means no more unnecessary executions just to check your files. or you can take to manually validate the cucumber syntax ,just Hit F9 .

Feature - Cucumber Quick Fix

From above validation your cucumber feature maybe met some warnings or errors ,so here we provide you to fix these errors. you can click the warning or error marker, it will show the options you can take to fix this error.

  1. For the cucumber step cannot be found issue in the workspace you can take to create a new step definition in a new file or you can take to create the step definition in the existing file.

  2. For the step definition in multiple files issue you can take to click the error warning marker ,it shows all these definitions files,if clicking this file ,it will navigate you to this file and highlight this duplicated step in file. This should save you much time .

Feature - Cucumber Content Assist(Auto Completion)

you will be prompted with a list of grammar valid cucumber step options or with a code template (Refer the code templates feature for more detail).

Feature - Cucumber step definition navigation (F3 or CTRL+Click)

Hit CTRL in the step definition, It will show the step you want to navigate and then just click it, the step definition file will be opened and the step being selected and highlighted,or you can take to use the eclipse shortcut F3 to achieve it .

Feature - Cucumber Code Template

Will allow people to really speed up feature scenario coding. It works as part of the code completion but it's user customizable: Just open your plugin preferences (Window > Preferences > Cucumber > Templates) and start messing with the default ones!

Feature - Cucumber Execution Runner

With This plugin you can run your .feature file from command line and see the execution report from cucumber console.

Feature - Cucumber Console

When you working on cucumber coding, all the executions or logs will be redirected to the cucumber console. You can know the cucumber progress well.

Feature - TestRail Integration

Yes, It supports integrate Cucumber with TestRail so you can check out or check in cucumber code between TestRail and Eclipse if you're using TestRail as cucumber script depository, this should make your TestRail code more easier.

Feature - Jenkins Integration

Yes, It supports integrate Cucumber with Jenkins now
